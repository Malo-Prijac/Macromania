Macromania

Ce programme est un site web de e-commerce.
On peut y trouver des jeux vidéos, des consoles et des accessoires.
Pour consulter le rapport de projet, veuillez lire le document ci-joint :
ALEXIS_SLOTA_PRIJAC_Rapport.pdf

#####################
# LANCEMENT DU SITE #
#####################

- 1) Aller dans le dossier bdd puis dans le fichier bddData.php. Ici, veuillez indiquer votre nom d'utilisateur et mot de passe SQL.
- 2) Avec les identifiants entrés ci-dessus, veuillez vous connecter à phpmyadmin et importer le fichier macromania.sql se situant dans le dossier sql.
- 3) Ouvrez votre serveur php en localhost
- 4) Lancer le fichier index.php dans un navigateur web (mettre en plein écran pour une meilleure visibilité)
- 5) Pour voir le fonctionnement de site vous pouvez naviguer dessus ou consulter le rapport.

PS : Pour revenir en arrière dans les pages web, l'utilisation des boutons mis à votre disposition sur la page web est préférée plutôt que d'utiliser ceux de votre navigateur.
